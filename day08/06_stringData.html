string : 가장 많이 다루는 데이터 타입

  • java에서 사용하는 string method와 거의 유사함.
  • 문자열 데이터는 index(0부터 시작)를 가지고 있음.
  • 문자열 값 내에서 (br) 개행하는 것을 추천하지 않음.
  • new String() : 객체 타입으로 선언 가능하지만 추천하지 않음.
  • length : 문자열의 총 길이
  • 추출관련 메서드 : substring(), substr(), slice()
  • 문자 객체 관련 : replace(old, new) => 처음 찾은 값만 사용가능
  • trim() : 양쪽 공백 제거
  • padStart(길이, 채울문자) / padEnd(길이, 채울문자) 길이만큼 문자를 채워주는 함수 950404-1****** / ***시
  • charAt(index) : 한글자 추출 / indexOf(문자) : 해당 문자의 위치 리턴
  • startWith(문자) / endWith(문자) 해당 문자열 앞, 뒤에 문자가 존재하는지 확인하는 매서드 return true / false
