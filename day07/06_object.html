Object : 객체

  • object : 데이터의 구조를 상세 특징으로 표현하려는 형식
  • const 로 선언하는 것이 일반적 : 객체의 구조를 보호. 지속성있게 유지 확보하기 위함.
  • {키(속성명): 값, 키:값, 키:값, ...}
  • 속성명 : key, 값 : value
  • 같은 Object 안의 같은 속성명은 불가능(key 중복불가능)
  • 객체 내부에서 속성값으로 사용될 수 있는 데이터는 제한이 없음.
  • 객체의 속성값으로 익명함수를 정의 = 메서드(method)라 칭함. 메서드의 이름은 속성명이 됨.
