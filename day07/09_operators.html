연산자

  • 산술연산자 : 사칙연산(+ - * /), 나머지(%), 제곱(**) 등
  • 대입연산자 : = += *= -= 자체값을 연산하여 다시 대입 a = b : a 변수에 b변수의 값에 대입 a += b : a 변수에 a+b 값을 대입 a = a + b
  • 비교연산자 : 이상(>=), 이하(<=), 초과(>), 미만(<), 같다(=), 다르다(!=) 비교연산자 결과는 항상 True / False
  • 논리연산자 : and(&&), or(||), not(!)
  • 증감연산자 : ++(1씩 증가), --(1씩 감소)
  • 비트연산자 : 웹 분야에서는 활용도가 낮음. 1111(2) && 11(2) = 3
  • 삼항연산자 : (조건식)? true:false
