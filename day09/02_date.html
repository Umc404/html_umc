날짜 관련 함수

  • new Date(); => 오늘날짜 반환 new Date(string); => string => 날짜 형식으로 변환
  • getFullYear(); 연도만 출력
  • getMonth()+1; => 0월부터 11월까지 반환하므로 함수 뒤에 +1 기입하여 출력
  • getDate(); => 1~31 리턴
  • getDay() => 요일을 번호로 리턴 0(일)~6(토) 0일 1월 2화 3수 4목 5금 6토
  • getHours(), getMinutes(), getSeconds() : 시분초 추출
  • toLocalDateString() => 2024. 08. 07 toLocaleTimeString() toLocaleString()
